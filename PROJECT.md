- **Репозиторий:** https://github.com/impusha3-ship-it/vmeste-anime (GitHub Pages, ветка `main`, корень)

## Стек / техника

Vanilla HTML + CSS + JS, без фреймворков. Ключевое:
- **SVG-фильтры** `feTurbulence` + `feDisplacementMap` (`#paint`, `#paintsoft`) — painterly «дрожащие» края (акварель/гуашь), не плоский вектор.
- Текстуры-оверлеи: зерно (`#g`), бумага-волокна (`#pap`), пигментные разводы (`.wash`, breathe-анимация).
- Атмосфера: god-rays, дымка (`.atmos`), виньетка.
- Шрифты (Google, все с кириллицей): **Comfortaa** (заголовки), **Nunito** (текст), **Marck Script** (акценты).
- Палитра (натуральная desaturated): церулеум `#5b9bc4`, шалфей/олива `#8aa86a`/`#5d7d4c`, персик `#e6b06a`, закат `#dd8f6b`, крем `#f4ecd6`.

## Экраны и разделы

- **Заставка** (`#splash`) при загрузке: солнце встаёт, облака въезжают, дух «попает», полоса загрузки. 3с (1.8с если уже залогинен), клик — пропустить.
- **Вход** → выбор пользователя (рисованные аниме-аватары Оли/Димы: большие глаза, румянец, причёски).
- **10 разделов** (как в оригинале): Календарь, Вместе, Вопрос дня, Тирлисты, Питомец, Письма, Фото, Музыка, Ачивки, Статистика.

## Оформление / фишки

### Фон и декор (Гибли)
- Многослойные дымчатые холмы + дальние купы деревьев, плывущие облака, качающаяся трава.
- **Саджи** (susuwatari) — летающие сажные духи; **кодамы** — два лесных духа у травы (головы покачиваются), ночью светятся.
- **Стая птиц** (две, машут крыльями) плывёт через небо.
- **Падающие лепестки сакуры** (18 шт).
- **Листовые уголки-украшалки** в углах экрана.

### Премиум-полировка
- Сочные кнопки: тройной градиент, внутренние блик/тень, shine-проводка по hover, glow, упругий press; `.btn.warm` (персик→закат).
- Карточки: многослойные тени, градиент-кант с золотым акцентом (mask-composite), стеклянный sheen, **tilt-параллакс** за курсором, **органичные «блоб»-радиусы** (неровные, не-шаблонные).
- Нав-бар: плавающая закруглённая «виджет-пилюля» (iOS-стиль) со стеклом; **рисованные SVG-иконки табов** (line-стиль).
- Появление карточек каскадом (stagger) при смене раздела; **brush-подчёркивание** заголовков (акварельный мазок, прорисовывается); рисованные волнистые разделители.
- Полароид-рамки фото (скотч, подпись 🍃, наклон, выпрямляются на hover).
- Кастомный градиент-скроллбар.

### Интерактив
- **Звук кликов** (WebAudio, без файлов) — модуль `Snd`: click/soft/tick/chime. Тумблер 🔊/🔇 (SVG), помнится.
- **Count-up** чисел (easeOutCubic) на статистике и счётчике дней + тиканье.
- **Параллакс-скролл** неба/облаков/холмов/разводов.
- **Ночной режим** 🌙 — луна, 90 мерцающих звёзд, 16 светлячков, перекраска UI; помнится. Контраст вычищен (нет светлого-на-светлом).
- **Фоновая мелодия** 🎵 — процедурная в духе Хисаиси (модуль `Music`: мажорная пентатоника + бас C-E-G-E + октавный shimmer), тихая; помнится, возобновляется по первому клику (autoplay policy).
- **Аниме-сердца** — переиспользуемый SVG-символ `#ah` (розовый/зелёный градиент, cel-обводка, блик, пульсация): на «Вместе» (5 шт) и бейджами на открытых ачивках.
- **Питомцы** (раздел Питомец, переключатель, выбор помнится): Мохнатик, **Тоторо**, **Кальцифер** (огонёк на полене, пламя мерцает), **Дзидзи** (чёрный кот). Все SVG нарисованы с нуля «по мотивам» — не копии кадров (копирайт чистый). Кнопка «Покормить» = подпрыгивание + аккорд.

### Иконки
- UI-иконки (10 табов + тумблеры music/night/sound/logout с состояниями) — **рисованные line-SVG** (`ICONS`/`svgIcon()`), не emoji.
- Намеренно оставлены emoji-акценты: иконки ачивок (по одной смысловой на каждую, 30 шт), 🍃/🍡 в кнопках, события календаря, splash.

### Доступность (по аудиту ui-ux-pro-max)
- `prefers-reduced-motion` — гасит ambient-анимации, сохраняя вид.
- `:focus-visible` кольца, `aria-label` на иконочных кнопках, `alt` у фото, поднят контраст текста.

## Данные (read-only из боевого проекта)

Firebase Realtime DB проект **`vmeste-a8b90`**, узел **`/app`** (тот же, что у боевого сайта — поэтому ТОЛЬКО чтение, чтобы не задеть).
REST без авторизации: `https://vmeste-a8b90-default-rtdb.firebaseio.com/app/<node>.json`

- **Важно:** значения хранятся как JSON-строки (двойная сериализация) — функция `fb()` парсит дважды.
- Узлы: `together_since` (`"2026-02-28"`), `ach` (obj `aN:bool`), `customAchs` (массив), `pet` (`{xp, claimed}`), `tracks`, `photos` (base64), `letters`, `questions` (по датам `YYYY-M-D` → `{d,o}`), `tierlists` (`weeks`), `events`, календарь `c_YYYY_M_D`.
- Каталог ачивок (id→иконка→название) и тема `fruits` тирлистов вшиты в файл (`ACHS`, `FRUITS`) — взяты из оригинала.
- Что показывается: дни вместе, посчитанная статистика, ачивки (открыто/впереди + кастомные), XP/уровень питомца, треки (ссылки), реальные фото, письма, последний вопрос дня, последняя неделя тирлиста.
